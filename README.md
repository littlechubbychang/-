# Stir-fried Minced Pork Car
## 1.人臉辨識：
>**註冊組員臉部相片，當車子偵測到組員的臉時，亮起LED燈**

## 2.聲音辨識：
>**當偵測到"yes"時，車子前進；當偵測到"no"時，車子後退；當偵測到拍手聲時，車子轉圈。**

## 3.物件偵測:
>**透過鏡頭看到畫面可以偵測出鏡頭裡的物體是什麼。**

## 4.訊息傳輸:
>**可將鏡頭所看到畫面傳輸至組員的LINE。**

## 5.機體遙控:
>**可以透過APP連結使用手機遙控車子行動。**
